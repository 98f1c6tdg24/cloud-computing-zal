# IaC-cloud-lab
Azure IaC lab: GitHub + Terraform + Ansible from manual provisioning to full automation.

This repository demonstrates three approaches to Infrastructure as Code (IaC):

1. Manual Terraform deployment
2. GitHub-based IaC workflow
3. Fully automated provisioning with shell scripts

The lab uses Azure CLI, GitHub, Terraform and Ansible to provision and configure a Linux virtual machine running Apache.
